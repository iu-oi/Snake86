# Snake86

a hungry snake game in masm 8086 assembly

Author: iu_oi

## compile & run

This program should run on a DOS.

`masm snake86.asm`

`link snake86.obj`

`snake86.exe`

## gameplay

Use W, A, S, D to steer your snake.

1 apple = 1 score = 1 body length

Hitting the wall or your body is fatal.

***HAVE FUN!!!***
